easyXDM is a javascript library that uses available techniques to provide a means of transporting messages and/or method calls between windows in different domains.
This is perfect if you plan to provide a client-side API on your web site as you can expose a method in as little as 7 lines of code.

In addition, you dont have to provide a full client library to the consumers, you can just tell them the location of your easyXDM-endpoint, and then show them the interface they will have to implement - and this can be as small as 1 line for a one-method interface!
